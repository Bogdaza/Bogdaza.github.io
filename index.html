Добро пожаловать на сайт!

На данной странице будет краткий экскурс по сайту.
    Вы сможете:
  • Посетить несколько страниц сайта: данную (домашнюю страницу), о причине создания сайта и контакты с создателем.
  • В дополнительном меню (самая правая кнопка) найти разделы, посвещенные HTML, CSS и JavaScript- языкам программирования.
  • На странице HTML можно понять как создавать каркас сайта.
  • В разделе CSS можно понять как придать сайту определенный стиль.
  • В разделе же JS можно научиться делать сайт функциональным (то есть чтобы он откликался на действия пользователя).
Так же хочется сказать, что следующие сайты были необходимы мне для создания и скорее всего пригодятся вам.
  1. W3schools - этот сайт содержит буквально все о различных языках, их структуре и свойствах.
  2. Shecodes - это сайт на котором есть множество полезной информации для работы с css, html и js.
  3. Stackoverflow - это крупный форум, где уже задано и решено много вопросов, с которыми можно столкнуться вовремя написания кода.
  4. Youtube - обязательно ютуб, где существует бессчисленное количество роликов по любой интересующей тебя теме.
